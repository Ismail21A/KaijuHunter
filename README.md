# KaizoKage — Communauté de collectionneurs de figures d’anime

## Nomenclature
- [objet] → Figure
- [inventaire] → Collection
- [galerie] → Galerie
- Utilisateur → Member

## Objectif
Gérer sa Collection privée de Figures et publier certaines Figures dans des Galeries visibles par la communauté.
